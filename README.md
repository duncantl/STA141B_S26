### Git

You can use git to fetch the current contents of the repository onto your machine.
There is a one-time "clone" of the repository and then  `pull` an any time
to get updates.



+ Make certain the git tool is installed

+ In a terminal, 

+ One time "clone" of the repository
```
git clone https://github.com/duncantl/STA141B_S26.git
```

   + This will create a directory STA141B_S26
   + This will contain the current snapshots of the files.

+ At any time, retrieve updates from the repository, from within the STA141B_S26 directory
```
git pull
```
